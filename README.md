# 📝 Task Manager (Console App)

A simple Java console-based Task Manager application to add, list, and remove tasks.

## 🚀 Features

- Add tasks
- View all tasks
- Remove tasks

## 🛠️ Requirements

- Java 8 or above
- Command-line terminal
